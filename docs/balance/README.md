# 基础平衡表导出

本目录用于存放自动导出的基础平衡数据，便于查阅与调参。

## 如何生成

```bash
npm run balance:dump
```

运行后会在当前目录生成：

- `balance.json`：机器可读的原始数据，包含关键常量、升级、研究、委托定义。
- `balance.md`：人类可读的表格形式，便于快速检视（含委托、外勤探索、学徒等）。

## 字段说明

- 关键常量：升华阈值、洞察公式常量、基础委托槽位、专注收益/冷却、离线时间上限等。
- 升级/研究：列出 ID、成本与加成类型。
- 委托：列出时长（秒）、奖励（科研/声望/精华）与接受所需的 EPS、声望要求。
- 学徒：列出原型基础资质、可选岗位、招募成本与各岗位的派生效果。
